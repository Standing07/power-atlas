# 🚀 部署指南（手把手教學，不用會寫程式）

這份教學帶你把「全球電力地圖」放到網路上，讓全世界都能瀏覽。全程免費，大約需要 30 分鐘。

整體流程：**把程式碼放上 GitHub → 用 Netlify（或 Vercel）自動建置上線**。之後每次程式碼或資料更新，網站都會自動重新部署。

---

## 第一步：申請 GitHub 帳號並上傳程式碼

GitHub 是全世界最大的程式碼儲存平台，也是本專案「開放原始碼」與「每月自動更新資料」的基地。

1. 到 https://github.com/signup 申請帳號（免費）。
2. 登入後，點右上角「＋」→「New repository」。
   - Repository name 輸入：`power-atlas`
   - 選 **Public**（公開，讓別人也能使用你的開放資料）
   - 其他都不要勾，按「Create repository」。
3. 打開你電腦的「終端機」（Terminal）App，貼上以下指令（把 `你的帳號` 換成你的 GitHub 帳號名稱）：

```bash
cd "/Users/sting/AI Projects/power-atlas"
git remote add origin https://github.com/你的帳號/power-atlas.git
git push -u origin main
```

4. 第一次推送會要求登入 GitHub——依照畫面指示用瀏覽器授權即可。
5. 重新整理 GitHub 頁面，看到一堆檔案就成功了 ✅

> 💡 如果 `git push` 出現權限錯誤，最簡單的解法是安裝 GitHub 官方工具：`brew install gh`，然後執行 `gh auth login` 依指示登入，再重新 `git push -u origin main`。

---

## 第二步：用 Netlify 上線（推薦，最簡單）

1. 到 https://app.netlify.com/signup ，選「**Sign up with GitHub**」直接用剛剛的帳號登入。
2. 點「**Add new site**」→「**Import an existing project**」→ 選「**GitHub**」。
3. 第一次會請你授權 Netlify 存取 GitHub——按「Authorize」，選擇 `power-atlas` 這個 repository。
4. 設定頁面 Netlify 會自動偵測到：
   - Build command：`npm run build`
   - Publish directory：`dist`
   - 都不用改，直接按「**Deploy**」。
5. 等 1–2 分鐘，畫面出現 🎉 就上線了！Netlify 會給你一個網址，例如 `https://random-name-123.netlify.app`。
6. 改成好記的網址：Site configuration → Site details → 「Change site name」，改成例如 `power-atlas`，網址就變成 `https://power-atlas.netlify.app`。

之後只要 GitHub 上的程式碼有更新（包含每月自動資料更新），Netlify 都會**自動重新部署**，完全不用再動手。

### 替代方案：Vercel

喜歡 Vercel 的話流程幾乎一樣：https://vercel.com/signup → 用 GitHub 登入 → 「Add New Project」→ 選 `power-atlas` → 按「Deploy」。專案已附好 `vercel.json` 設定，不用改任何東西。

---

## 第三步：確認每月自動更新資料有開啟

1. 到你的 GitHub repository 頁面，點上方「**Actions**」分頁。
2. 如果看到黃色提示，按「**I understand my workflows, go ahead and enable them**」。
3. 完成！之後每月 5 日，GitHub 會自動：下載最新的 OWID／IRENA 資料 → 通過驗證 → 提交 → 網站自動重新部署。
4. 想立刻手動更新一次：Actions → 左邊選「Update data」→ 右邊「Run workflow」。

> 🛡️ 資料驗證失敗時（例如上游改了格式），更新會中止、網站保持舊資料不會壞，你會在 Actions 看到紅色叉叉——到時候再處理即可（或回來找 Claude 幫忙）。

## 之後想改內容怎麼辦？

| 想做的事 | 方式 |
|---|---|
| 更新台灣用電大戶、企業綠電數字 | 編輯 `src/data/taiwan-detail.json`、`src/data/companies.json`（記得同步更新來源與年份），存檔後 `git add -A && git commit -m "update data" && git push` |
| 立刻抓最新國際資料 | 在專案資料夾執行 `npm run data -- --force`，然後 commit + push |
| 在本地電腦先預覽 | `npm run dev`，瀏覽器開 http://localhost:5173 |
| 買自己的網域（選配） | 在 Netlify：Domain management → Add a domain（網域一年約 NT$300–600） |

## 常見問題

**Q：這些服務真的免費嗎？**
GitHub 公開 repository 免費；Netlify/Vercel 免費方案的流量對這種公益網站綽綽有餘。唯一可能花錢的是自訂網域（選配）。

**Q：別人要怎麼使用我的開放資料？**
你的網站上線後，`https://你的網址/data/countries.json` 就是免費 API。詳細欄位說明在網站的「關於與開放資料」頁。

**Q：網站壞了怎麼辦？**
Netlify 的 Deploys 頁面可以一鍵「Rollback」回到之前任何一個正常版本。
